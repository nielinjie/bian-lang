# 第三阶段：如果哎

本阶段的任务是实现控制流。以 if/else 为例。

## 语法

```
if (a) {
    b = 1
} else {
    b = 2
}
```

if/else 控制流语句，举例如上。
几个要点：

1. 条件表达式，由括号包围。
1. 分支，由大括号包围。
1. 条件表达式，目前没有数据类型，都是整数。所以就是以非零作为真，零作为假。

## 开发

1. `Block` 是一个新的结构单位，由一个或多个`Statement`组成。
1. if/else 的两个分支，分别是一个`Block`。
